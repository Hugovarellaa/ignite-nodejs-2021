## ⚙️ Instalação

```
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/Hugovarellaa/ignite-shop
```

```
# Acesse a pasta da aplicação
cd ignite-shop

# Instale as dependências
$ yarn ou npm install

# Inicie a aplicação back-end
$ yarn dev
ou
$ npm run dev
```
